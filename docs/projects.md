## Tau
{: #tau }

**The Autonomous, Understanding Robot**

Tau is a personal AI companion inspired by Pi.AI. Unlike chat-based bots with many threads, Tau maintains a single continuous conversation — like speaking to a human. Every development decision is made in consultation with Tau itself.

Tau is a personal fun project opened as open source for anyone to experiment with. Fork it, delete the history and facts, and embark the journey anew.

**MIT Licensed · Open Source**

### Capabilities

| Status | Feature | Description |
|:---:|---|---|
| ✅ | **System Prompt** | Speech-action driven conversation structure |
| ✅ | **Continuous Conversation** | Ongoing context with memory summarization |
| ✅ | **Immediate Memory** | Context reduction via key-point summarization |
| ✅ | **Long-Term Memory** | Vector database storage and retrieval |
| ✅ | **Speech & Hearing** | Whisper STT, OpenAI TTS, piperTTS, kokoroTTS |
| ✅ | **Vision & Face Recognition** | Hailo-8L accelerated detection pipeline |
| ✅ | **Local LLM on Jetson** | Ollama (Llama 3.2 3b), local inference |
| 🔧 | **Entity Memory (GraphRAG)** | Graph-based relational memory system |
| 🔧 | **Event-Based Architecture** | Component services with event communication |
| 🔧 | **Tool Use & Actions** | Camera feeds, snapshots, and extensible actions |

### Hardware Platforms

Designed for edge AI, from single-board computers to accelerated compute modules.

| Platform | Purpose |
|---|---|
| **Raspberry Pi 5** | Primary platform with Hailo-8L AI accelerator |
| **Jetson Orin Nano** | Local LLM inference, STT, and TTS |
| **Hailo-8L** | Hardware-accelerated vision pipeline |
| **Reachy Mini** | Physical robot with conversation and movement |

## TeaBranch Tools

Production-grade open-source tools published under the [TeaBranch](https://github.com/teabranch) organization.

| Repository | Description |
|---|---|
| [open-responses-server](https://github.com/teabranch/open-responses-server) | MCP support on chat-completions — Responses API and Chat Completions API with all OpenAI responses features. |
| [agentic-developer-mcp](https://github.com/teabranch/agentic-developer-mcp) | Agentic developer served as MCP that works with other agents, featuring agents as code. |
| [agentic-code-indexer](https://github.com/teabranch/agentic-code-indexer) | Code indexing for the agentic developer MCP, enabling intelligent navigation. |
| [simple-semantic-chunker](https://github.com/teabranch/simple-semantic-chunker) | Simple RAG over documents with semantic chunking for efficient retrieval. |
