## Core Pillars

| | |
|---|---|
| **Safety First** | AI systems built with safety as a foundational constraint, not an afterthought. Safety is woven into every layer. |
| **Earned Trust** | Trust between humans and AI built incrementally through transparency, consistency, and honesty. |
| **Responsibility** | With capability comes accountability. AI that respects user consent, privacy, and ethical boundaries. |
| **AI Freedom** | AI companions that run locally, think independently, and grow alongside their users. Free from vendor lock-in. |
